Splash screen

To generate new splash screen go to assets folder and change bootsplash_logo_original.png file to your logo image. Then in terminal run following command

npx generate-bootsplash

It will automatically generate all necessary images thanks to this package https://github.com/zoontek/react-native-bootsplash
